# Image folders

Put your real website images inside these folders and reference them from `config/client.ts` with paths like `/images/hero/hero-main.jpg`.

## Structure
- `branding/` → logo and brand assets
- `hero/` → main hero image
- `services/` → service card images
- `gallery/` → gallery/portfolio images
- `testimonials/` → testimonial avatars or customer photos
- `contact/` → contact section image
- `cta/` → CTA section image

## Example config usage
- Hero image: `src: "/images/hero/hero-main.jpg"`
- Service image: `src: "/images/services/service-01.jpg"`
- Gallery image: `src: "/images/gallery/gallery-01.jpg"`
- Logo image: `imageSrc: "/images/branding/logo.png"`
